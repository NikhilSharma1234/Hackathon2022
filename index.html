Hackathon 2022

Receiving Emails and Storing to Directory

This portion of the code receives the last three emails from our test email 'hackathontest22@gmail.com' and email and stores them to the local directory for processing. We have the user input email credentials through the terminal, only moving on of the credentials are authorized. After fetching an email, the program decodes it to check if there is an attachment and if so stores the attachment separately in a different directory. For the rest of the email, a .txt file is created and printing begins with email that the message was received from, followed by the subject and finally the body of the email.

Spam Email Filtering System

Returning Latest Email to Spam Email Folder

This portion of the program moves the last received email from the inbox to the spam. We first have the user input their email credentials and only proceeds if the credentials are authorized. If so, we move on to specify that we are taking the last received email from the 'INBOX', copying it, and then storing the copy in the 'Spam' folder.
